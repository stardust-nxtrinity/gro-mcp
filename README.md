# @gro/mcp

Your AI just got a full marketing team.

Connect Gro to Claude, Cursor, or Windsurf in two minutes. Then just describe what you want — Gro handles the research, creative, and launch.

## Install

```bash
npx @gro/mcp install
```

Auto-detects Claude Code, Claude Desktop, Cursor, and Windsurf. Sign in with your Gro account when the browser opens. That's it.

## What you can do once connected

- **Image ads** — describe what you need, get variants back with copy written and ready to launch
- **Video ads** — storyboard to animated video in one conversation
- **Launch to Meta** — campaign structure, targeting, budget, live or draft
- **Research hooks** — sweeps TikTok, YouTube, and the Meta Ad Library for what's converting right now
- **Ad performance** — ROAS, CPM, winning patterns pulled from your Meta account
- **Product setup** — paste a URL, Gro reads it and builds the product record

Tools are discovered automatically when you connect — nothing to configure.

## Requirements

- Node.js 18+
- A Gro account — [gro.app](https://gro.app)
- Claude Code, Claude Desktop, Cursor, or Windsurf

## Staging

```bash
npx @gro/mcp install --staging
```

Connects to `test-api.gro.app` instead of `api.gro.app`. Installs as `gro-staging` so it coexists with prod.
